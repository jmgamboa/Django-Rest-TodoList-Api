Django Rest Todo List Api Doc

Todo Lists
Path: /api/todo | METHODs = GET, POST | Users' Todo Lists
Path: /api/todo/{:id} METHODS = POST, PUT, DELETE | Single Todo List View By Todo List ID
Todo Items
Path: /api/todoitem | METHODS = GET, POST | Users' Todo Items
Path: /api/todoitem/{:id} METHODS = POST, PUT, DELETE | Single Todo list Item By todoitem ID
Registration and Authentication
Path: /user/register METHODS = POST, Format=JSON, ex. {"username": "John Doe", "email": "John@gmail.com", "password": "abc"}
Path: /user/login METHODS = POST, Format=JSON, ex. {"username": "John Doe", "password": "abc"}
Path: /user/logoutMETHODS = GET | logs out current session
